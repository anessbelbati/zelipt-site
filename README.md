# zelipt-site (Astro)

Bare Astro scaffold for the Zelipt marketing site.

- No styling
- Plain placeholder content only

## Pages

- `/` → `src/pages/index.astro`
- `/pricing` → `src/pages/pricing.astro`
- `/contact` → `src/pages/contact.astro`

## Development

From this directory:

```sh
npm install
npm run dev
```

Build / preview:

```sh
npm run build
npm run preview
```
